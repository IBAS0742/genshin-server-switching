# genshin-server-switching
Windows版原神大陆地区bilibili/官服切换程序

## 如何使用
将编译完成的文件夹里面的文件复制到你的原神文件夹，然后就可以使用啦！
（ps：此程序是方便大家移动config.ini文件，本人到时候也会添加切换服务器后直接打开程序）
![移动文件夹](截图/move.png)<br>
然后就直接创建快捷方式到桌面即可！<br>
（因为是用vscode编译的，所以没有添加icon）<br>
![快捷方式](截图/kuaijie.png)
