# genshin-server-switching
Windows版原神大陆地区bilibili/官服切换程序

## 如何使用
将编译完成的文件夹里面的文件复制到你的原神文件夹，然后就可以使用啦！
（ps：此程序更改完服务器后会延迟5秒后打开原神qwq）
![移动文件夹](截图/move.png)<br>
然后就直接创建快捷方式到桌面即可！<br>
![快捷方式](截图/kuaijie.png)
